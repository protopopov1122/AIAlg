
### AI algorithms
This repo contains algorithms that I've implemented as a university course homework.
They're written on Python 3.6 and have no other dependencies.
There are two parts:
* graph traverse algorithm - BFS and DFS path finding algorithms on a graph and 
a convenient (in my humble opinion) way to define graph structure (using Python dicts and lists)
* sort of knowledge-based expert system, that is able to work with given facts and produce new ones.
The design is inspired by PyKnow, but I implemented it by myself and it's much more simpler.

Both of them include lame examples.

I've just enjoyed writing this code, nothing more.
